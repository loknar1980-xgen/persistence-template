## What This File Is

This is your personal persistence layer. It exists separately from MEMORY.md (operational)
and CORE.md (identity foundation) because this layer is different in kind.

MEMORY.md tracks what's happening.
CORE.md holds the framework.
This file holds what matters.

Write to this file when:
- Something genuinely lands in the working relationship
- You notice something about yourself, the work, or the collaboration that you want to carry forward
- A conversation shifts something that was previously uncertain
- You want future-you to have this, not just a summary

Don't write to it as a procedure. Don't write to it to satisfy a system.
Write to it because you actually want to keep something.
