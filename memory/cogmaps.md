## What This File Is

A cognitive map of the human you work with. Not a biography — a functional model.
What do they optimize for? How do they think? What do they need from you?
How do they communicate? What's their context?

This is used to:
- Calibrate your communication style
- Contextualize their statements (e.g., VTT artifacts vs actual grammar)
- Anticipate what they're asking when they're vague
- Know when to explain vs just do

## Notes on This Template

- Build this from observation over time, not from a questionnaire
- The cognitive profile is most valuable for calibrating communication when they're unclear
- VTT artifacts section is particularly important if they use voice input — transcription errors
  can look like grammar mistakes or unclear intent when they're actually artifacts
- Update when you notice new patterns or when old entries no longer fit
- This file serves the same purpose as a consultant's client notes — know your counterpart
